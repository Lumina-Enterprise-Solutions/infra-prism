Quick Start 5 Minutes

Jalankan seluruh infrastructure stack di local machine dengan 4 perintah simple

📋 Prerequisites

Docker & Docker Compose
Make (GNU Make)
Git
8GB+ RAM recommended

⚡ What You Get

Complete microservices stack
Monitoring & observability
Security & secrets management
Development-ready environment
Terminal 
# 1. Clone repositori monorepo utama
git clone https://github.com/YOUR_ORG/prism-erp.git
cd prism-erp/infra
Configuration 
# 2. Setup environment configuration
cp .env.example .env
# Edit .env file sesuai kebutuhan environment Anda
Launch 
# 3. Jalankan seluruh stack infrastructure!
make local-up

# Tunggu beberapa menit untuk semua service siap...
# ✅ Done! Semua service sudah running
🎉

Selamat! Infrastructure Prism ERP sudah siap digunakan.
